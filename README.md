Code repository for the paper "The Influence of Choices of Prior Information on Optimal Designs of Seismic Source Location Surveys" by Dominik Strutz, Tjeerd Kiers, Cedric Schmelzbach, Hansruedi Maurer, and Andrew Curtis.

The repository is currently only a filler and will be updated in the near future. The final version of the code will be made available upon publication of the paper.
