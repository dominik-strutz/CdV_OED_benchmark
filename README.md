Code repository for the paper "The Influence of Choices of Prior Information on Optimal Designs of Seismic Source Location Surveys" by Dominik Strutz, Tjeerd Kiers, Cedric Schmelzbach, Hansruedi Maurer, and Andrew Curtis.

If you us the data (travel time picks or velocity models) in this repository, please cite the following paper:
> Kiers, T., Schmelzbach, C., Maurer, H., Amann, F. & Robertsson, J., 2026. Imaging one of the largest Alpine slope instabilities with 3D seismic first-arrival traveltime tomography. J. Appl. Geophy., 247, 106081, Elsevier BV.
